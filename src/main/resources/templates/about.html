Данный ресурс предназначен для изучения английского языка и представляет собой таблицу времен, которую можено заполнять фразами с переводом. Это тренажер для закрепления знаний и повторения пройденного материала.

Так же - это учебный проект для закреплений знаний по разработке на языке Java. Подробнее:

  • проект построен на Spring Boot
  • сборщик проекта - Maven
  • авторизация - Spring security
  • шаблонизатор - Thymeleaf
  • база данных - Postgresql
  • миграции базы данных - FlyWey
  • так же: lombok, hibernate-validator
